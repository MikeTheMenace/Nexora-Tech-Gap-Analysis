# Nexora Tech Solutions – Cybersecurity Gap Analysis


## Detailed Gap Analysis

| Function | Current State Summary | Gap Severity | Key Recommendations |
|----------|----------------------|--------------|----------------------|
| **GOVERN** | Informal, reactive governance. No risk appetite. Third‑party risks not assessed. | **High** | Appoint CISO, define risk appetite, implement vendor risk program. |
| **IDENTIFY** | Incomplete asset inventory. No formal risk assessment methodology. Passive threat intel. | **High** | Deploy asset management, adopt structured risk assessment (e.g., FAIR), operationalise threat intelligence. |
| **PROTECT** | IAM inconsistent. Backups exist but not tested. Legacy config drift. | **Medium** | Automate privileged access reviews, test backups quarterly, harden legacy systems. |
| **DETECT** | No centralised logging. Manual alert review. No correlation. | **High** | Deploy SIEM, define alert thresholds, establish continuous monitoring. |
| **RESPOND** | No documented IR plan. Informal tracking. Incomplete root cause analysis. | **High** | Write and test incident response plan, use incident ticketing, mandate RCA for all major incidents. |
| **RECOVER** | Partial recovery plans. No BIA. Untested backups. | **Medium** | Complete business impact analysis, define recovery priorities, conduct tabletop exercises quarterly. |

## Reccomendations

### Short‑term (0–3 months)
1. Appoint CISO
2. Define  risk appetite and set policies
3. Add cyber risk to ERM
4. Ensure visibility of all devices in environment

### Medium‑term (3–6 months)
4. Create an asset Directory and active directory for all assets
5. Implement SIEM with defined alert thresholds
6. Review and assess all alerts weekly.
7. Implement a playbook and repoting matrix

### Long‑term (6–12 months)
7. Formalise third‑party risk management program
8. Automate backup integrity testing
9. Achieve NIST CSF Tier 3 across all functions
